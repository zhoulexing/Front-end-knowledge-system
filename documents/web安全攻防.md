## XSS 跨站脚本攻击

1. 非持久性 XSS（反射型 XSS）
   一般是通过给别人发送带有恶意脚本代码的 URL，当 URL 被打开时，特有的恶意代码参数被 HTML 解析、执行。

特征：

-   即使性，不经过服务器存储；
-   攻击者需要诱骗点击，必须通过用户点击链接才能发起；
-   反馈率低，所以较难发现和响应修复；
-   盗取用户敏感保密信息。

防止：

-   页面渲染的所有内容或数据必须来自服务器端；
-   尽量不要从 URL、document.referrer、document.forms 等这种 API 中获取数据渲染；
-   前端渲染的时候对任何的字段都需要做 escape 转义编码。
